# **$isJSON**
> **Checks is a valid JSON** <br/>
> $isJSON[object;hjson?]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Object | The object to validate | HJSONEncodable &#124; JSONEncodable | True |
| Hjson | Whether or not to use HJson | Boolean | False |

### Returns
> Unknown
