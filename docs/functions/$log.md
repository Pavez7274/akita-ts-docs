# **$log**
> **Log something in console** <br/>
> $log[title;...stuffs]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Title | Unknown | String | False |
| ...stuffs | Unknown | Any | True |

### Returns
> Void

### Example
> ```php
$log[;hi]
```
