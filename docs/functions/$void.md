# **$void**
> **Make a void return** <br/>
> $void[sentence]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Sentence | Unknown | String&lt;interpretableCode&gt; | True |

### Returns
> Void

### Example
> ```php
$void[$var[hi;asdf]] // creates the variable hi, but does not return its value
```
