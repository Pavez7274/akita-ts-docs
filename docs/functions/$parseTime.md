# **$parseTime**
> **Convers time to miliseconds** <br/>
> $parseTime[time]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Time | Unknown | String&lt;timeResolvable&gt; | True |

### Returns
> Number

### Example
> ```php
$parseTime[1s1h] // 3601000
```
