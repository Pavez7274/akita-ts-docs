# **$lower**
> **Converts string to lower case** <br/>
> $lower[text]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Text | Unknown | String | True |

### Returns
> String

### Example
> ```php
$lower[HI] // hi
$lower[var:NAME] // convers string value to lower case
```
