# **$kebab**
> **Converts string to kebab case** <br/>
> $kebab[text]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Text | Unknown | String | True |

### Returns
> String

### Example
> ```php
$kebab[I LOVE KEBAB STYLE] // i-love-kebab-style
```
