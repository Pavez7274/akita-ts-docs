# **$send**
> **Send the instance** <br/>
> $send[returnId]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| ReturnId | Whether or not to return message Id `(default: no)` | Boolean | True |

### Returns
> ReturnId ? Snowflake : Void

### Example
> ```php
$send[no]
```
