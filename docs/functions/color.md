# **$color**
- - -
> **Sets the color of this instance** <br/>
> $color[index;colorResolvable]

### Fields
> | Name          | Type                | Optional |
|-----------------|---------------------|----------|
| index           | number[0,4]         | false    |
| colorResolvable | string<hex \| name> | false    |

### Returns
> Void

### Example
> ```php
$title[0;Embed 0]
$description[15 minutos con vos! ouh-oh-oh!]
$color[0;Red]
$send[no]
```
