# **$capitalize**
> **Converts the first character of string to upper case and the remaining to lower case** <br/>
> $capitalize[text]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Text | Unknown | String | True |

### Returns
> String

### Example
> ```php
$capitalize[hi HaTE MySElf] // I hate my self
```
