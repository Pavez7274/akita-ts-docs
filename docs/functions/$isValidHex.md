# **$isValidHex**
> **Check if the provided value is a valid hex code** <br/>
> $isValidHex[resolvable]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Resolvable | Unknown | String | True |

### Returns
> Boolean

### Example
> ```php
$isValidHex[#124f09] // true
```
