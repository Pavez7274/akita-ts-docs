# **$randomNumber**
> **Produces a random number between min and max** <br/>
> $random[min;max;decimals?]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Min | Unknown | Number | True |
| Max | Unknown | Number | True |

### Returns
> Any

### Example
> ```php
$randomNumber[0;400] // random number between 0 and 400
```
