# **$last**
> **Gets the last element of array** <br/>
> $last[key]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Key | The key of array or string | String&lt;variable&gt; | True |

### Returns
> String

### Example
> ```php
$var[str;hi, im a string] // string var
$var[arr;[3,5,2,7,2]] // array var
$last[str] // g
$last[arr] // 2
```
