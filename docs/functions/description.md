# **$description**
- - -
> **Sets the description of this instance** <br/>
> $description[index;text]

### Fields
> | Name | Type        | Optional |
|--------|-------------|----------|
| index  | number[0,4] | false    |
| text   | string      | false    |

### Returns
> Void

### Example
> ```php
$content[0;Interpol - The New!]
$description[*"You looking alright tonight... I think should be go"*]
$send[no]
```
