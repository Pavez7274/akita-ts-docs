# **$setImage**
> **Sets the image of this embed** <br/>
> $setImage[index;url]
- - -

### Fields
| name | description | type | required |
|------|-------------|------|----------|
| Index | The index of the embed | Number[0,4] | True |
| Url | The image url of the embed | String&lt;URL&gt; | True |

### Returns
> Void

### Example
> ```php
$setImage[0;https://discord.com/]
```
